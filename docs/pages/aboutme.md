# 关于我

名字的来源是烟雨江南的《罪恶之城》，虽然现在看这本小说，会发现有很多缺点，但流砂这条线写的还是可以的，尤其是反转的时候。

因为习惯且喜欢记录，所以搭了这个博客，也算是我的一个精神家园，也取名 Flowsand Cabin。
